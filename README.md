# 飞书热榜&智能效率助手

## 项目简介
一款基于飞书CLI开发的**AI驱动型热榜查询工具**，融合多领域榜单查询、飞书多维表格联动、AI智能答疑三大核心功能，轻量高效、开源可复用，完全贴合飞书CLI大赛“AI Agent接管办公、高实用、可落地”的评审标准，致力于打造办公场景下的高效热榜信息工具，具备较强的冲奖竞争力。

### 1. 技术亮点
- ✅ AI Agent原生集成：基于轻量LLM模型，实现意图精准识别、智能答疑、榜单解读，符合大赛核心导向
- ✅ 飞书生态深度联动：支持榜单数据一键导出至飞书多维表格，自动生成可视化图表，贴合飞书办公场景
- ✅ 多领域精准覆盖：支持综合、科技、娱乐、财经、职场5大领域，避免同质化，满足不同用户需求
- ✅ 鲁棒性强：具备错误自动修复机制，网络异常、操作失误时可给出明确修复建议，评委测试体验佳

### 2. 实用亮点
- 🚀 轻量化部署：无复杂依赖，Node.js环境一键启动，评委可快速测试
- 📱 富文本展示：支持榜单美化、重点标注，视觉体验优于纯文本Skill
- 🔍 精准意图识别：支持模糊查询、多关键词组合，用户使用门槛低
- 📊 数据可复用：导出的表格可直接在飞书编辑、分享，提升办公效率

### 3. 参赛合规亮点
- 📄 开源合规：MIT协议，代码完全公开，可自由复用、修改
- 📋 规范适配：严格遵循飞书CLI Skill开发规范，支持text、rich_text、card等多种展示方式
- 🛡️ 安全无敏感：采用公开合规的榜单接口，无隐私数据采集，无安全风险

## 功能介绍
| 功能模块 | 具体能力 | 触发关键词示例 |
|----------|----------|----------------|
| 多领域榜单查询 | 查询5大领域热榜，支持热度排序、精准筛选 | 查科技热榜、娱乐榜单排名、财经热门 |
| AI智能答疑 | 解读榜单、对比榜单、分析热度趋势 | AI解读科技热榜、哪个榜单热度最高 |
| 飞书表格联动 | 一键导出榜单至飞书多维表格，自动生成图表 | 导出综合热榜到表格、表格导出 |
| 错误自动修复 | 网络异常、操作失误时给出修复建议 | （自动触发，无需额外关键词） |

## 技术栈
- 核心框架：Node.js + 飞书CLI
- AI能力：llama-index（轻量LLM，无需复杂部署）
- 飞书生态：feishu-base-api（多维表格联动）
- 网络请求：axios（稳定高效）

## 快速部署步骤
1. 环境准备：Node.js 16+、飞书开放平台应用（获取appId/appSecret）
2. 安装依赖：npm install
3. 配置环境变量：新建.env文件，添加FEISHU_APP_ID、FEISHU_APP_SECRET
4. 启动测试：npm start
5. 飞书触发：在飞书输入关键词（如“查科技热榜”），即可使用
